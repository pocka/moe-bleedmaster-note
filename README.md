### コマンド一覧

| 概要                       | コマンド                                    | 備考 |
| -------------------------- | ------------------------------------------- | ---- |
| Node.jsのバージョン合わせ  | `nvm install`(初回) or `nvm use`            |
| 依存パッケージインストール | `npm i`                                     |
| 開発環境の起動             | `npm start`                                 |
| テスト                     | `npm t`                                     |
| コミットテンプレートの登録 | `git config commit.template .commitmessage` |
